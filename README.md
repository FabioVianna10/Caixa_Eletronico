#  Projeto  Caixa_eletronico

 # Criado um sistema inspirado nas funcionalidades de um caixa eletônico, com as opções:
 # Criar uma conta;
 # Depositar;
 # Consultar saldo;
 # Sacar;
 # Sair da conta;
 
 
 
 # Sistema criado com NODE JS, módulos:  Inquire, Chalk e FS
